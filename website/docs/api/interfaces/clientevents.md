### roomChange

•  **roomChange**: (after: [Room](../classes/room.md), before: [Room](../classes/room.md)) => void

Emitted when the room is changed.

**`example`** 
```js
client.on('roomChange', (after, before) => {
	console.log('The room changed from '+before.name+' to '+after.name);
})
```
